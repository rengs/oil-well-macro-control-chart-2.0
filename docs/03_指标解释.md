# 指标解释

## supply_capacity_index

供给能力指数，综合注采支持、动液面、泵效和液量供给状态构造。数值越高，表示液量供给能力越强。

## production_demand_index

生产需求指数，综合日产液、日产油、含水和生产天数构造。数值越高，表示当前生产系统对供给的需求越强。

## supply_production_ratio

供采匹配比，计算公式为：

```text
supply_production_ratio = supply_capacity_index / production_demand_index
```

## energy_intensity_kwh_per_t

吨油能耗，表示单位产油消耗的电量。该指标用于识别能效异常井。

## measure_potential_score

措施潜力评分，综合供采偏差、能耗、泵效、含水和近期维护频次构造。分数越高，越适合作为措施优先候选。

## control_zone

基于供采匹配比划分：

- `short_supply`: 供给偏弱
- `balanced`: 供采均衡
- `excess_supply`: 供给偏强
