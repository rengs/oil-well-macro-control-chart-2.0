# 制作流程

## 1. 数据准备

收集单井日产液、日产油、含水、液面、压力、泵效、能耗、测试频次和维护记录等字段。

## 2. 指标计算

运行：

```bash
python src/calculate_indicators.py --input data/sample_wells_500.csv --output data/sample_wells_500_with_indicators.csv
```

## 3. 图件生成

运行：

```bash
python src/plot_macro_control_chart.py --input data/sample_wells_500.csv --output-dir examples --asset-dir assets
```

## 4. 结果解释

结合 `docs/03_指标解释.md` 中的指标定义，对供采偏差、能耗异常和措施潜力进行解释。

## 5. 现场校准

正式应用前，需要用历史措施效果、区块开发方案和现场专家规则校准阈值。
